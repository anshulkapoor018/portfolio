# Anshul Kapoor Portfolio

Personal portfolio built with Next.js, TypeScript, Tailwind CSS, and `next-themes`.

## Overview

This portfolio includes:
- Responsive light and dark themes
- Resume-aligned hero, experience, education, skills, and projects sections
- Keyboard-accessible theme switching and link focus states
- Production build and lint scripts

## Local Development

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Production Check

```bash
npm run lint
npm run build
```
